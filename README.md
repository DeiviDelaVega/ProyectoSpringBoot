# ProyectoSpringBoot
Este proyecto es una aplicación web desarrollada con **Spring Boot**, que permite gestionar inmuebles mediante operaciones CRUD (Crear, Leer, Actualizar, Eliminar). El sistema está conectado a una base de datos **MySQL** y utiliza **JPA** e **Hibernate** para el acceso a datos, y **Thymeleaf** como motor de plantillas para la capa de presentación.
